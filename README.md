# Estrutura de Dados I

### Estrutura de Diretórios

O material está organizado pelos tópicos de estudo e blocos de avaliações da disciplina:

* **Arrays e Estruturas:** Introdução à organização de dados utilizando vetores e structs.
* **Lista Dinâmica:** Implementações de listas encadeadas e manipulação de ponteiros.
* **Pilha:** Criação e gerenciamento da estrutura de dados do tipo Pilha (LIFO - Last In, First Out).
* **ESTUDOS P1, P2 e P3:** Resumos, exercícios de fixação e códigos focados na preparação para as provas da disciplina.
